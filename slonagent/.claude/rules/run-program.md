# Запуск программы

Всегда запускать через виртуальное окружение `.venv` и `start.bat`.

Для CLI-режима:
```
start.bat --cli
```

## Виртуальное окружение

Проект использует `.venv` в корне. Никогда не запускать через системный Python.

Перед запуском убедиться, что активировано `.venv`:
```
.venv\Scripts\activate
```
